# Contributor Covenant Code of Conduct

## Our Pledge

We as members, contributors, and maintainers pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

- Be respectful and considerate.
- Welcome constructive feedback.
- Focus on what is best for the project and community.
- Do not engage in harassment, discrimination, or exclusionary behaviour.

## Enforcement

Report unacceptable behaviour to the project maintainers through private contact methods available in this repository.

All reports are reviewed and investigated promptly and fairly.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, discussions, and related community channels.
